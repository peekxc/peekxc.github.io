Software

  • primate

    Provides estimators of quantities derived matrices whose spectra are parameterized by functions.

    f(A)≜Uf(Λ)U,f:[a,b]→Rf(A) \triangleq Uf(\Lambda)U, \quad f : [a,b] \to \mathbb{R} f(A)≜Uf(Λ)U,f:[a,b]→R

    By choosing different parameterizations of fff one can approximate a variety of spectral quantites, including e.g. the numerical rank, the log-determinant, the trace inverse, Schatten norms, the trace of matrix exponentials, the eigencount, and the spectral density.

    Applications of matrix functions include characterizing folding in proteins (Estrada index), principal component regression, spectral clustering, Gaussian process likelihood estimation, characterizing graph characteristics, and deep neural loss landscape analysis.

  • simplextree

    simplextree is joint R / Python package that simplifies computation for general simplicial complexes of any dimension by providing pybind11 bindings to a Simplex Tree data structure implemented in modern C++17. Natively supports many tree operations, including insertions, removals, expansions, collapses, star/link enumerations, and various ordered traversals.
