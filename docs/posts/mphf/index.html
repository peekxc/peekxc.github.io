A perfect hash function is an injection h:S→Mh : S \to Mh:S→M, where m=∣M∣m = \lvert M \rvertm=∣M∣ must necessarily satisfy m≥nm \geq nm≥n. If m=nm = nm=n and M={0,1,…,n−1}M = \{0, 1, \dots, n - 1\}M={0,1,…,n−1}, hhh is called a minimal perfect hash function (MPHF).
